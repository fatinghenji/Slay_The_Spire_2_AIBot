## 游戏概述

《杀戮尖塔2》是一款融合了卡牌构建与Roguelike元素的策略游戏。玩家需要选择角色，攀登尖塔，与各种敌人战斗。

## 角色介绍

### 1. 铁甲战士 (The Ironclad)

| 属性 | 值 |
|------|-----|
| 称号 | 坚韧的战士 |
| 基础HP | 75 |
| 初始遗物 | Burning Blood (燃烧之血) |

**特点：**
- 擅长力量叠加，每点力量增加所有攻击牌的伤害
- 拥有自我治疗能力
- 消耗机制：消耗牌组中的牌获得额外效果
- 多段攻击配合力量收益极高

**核心关键词：** 力量(Strength)、消耗(Exhaust)、生命回复(Heal)

**推荐构建：**
1. **力量成长流** (S级, 新手友好)
   - 核心思路：叠加力量使每张攻击牌造成巨大伤害
   - 关键卡牌：Demon Form(恶魔形态)、Limit Break(突破极限)
   
2. **消耗引擎流** (A级, 中等难度)
   - 核心思路：消耗牌组触发强力协同
   - 关键卡牌：Feel No Pain(无痛)、Dark Embrace(黑暗拥抱)、Barricade(壁垒)

3. **自伤狂战流** (A级, 高级难度)
   - 核心思路：牺牲生命值获得强力效果
   - 关键卡牌：Rupture(裂伤)、Brutality(残暴)、Offering(祭品)

2. **分身流** (A级)
   - 核心思路：生成大量分身牌进行高频攻击
   - 关键卡牌：Blade Dance(剑舞)、Accuracy(精准)

3. **弃牌流** (A级)
   - 核心思路：弃牌触发效果并循环利用牌组
   - 关键卡牌：Tactician(战术家)、Reflex(反射)

## 卡牌系统

### 卡牌类型

| 类型 | 说明 |
|------|------|
| 攻击 (Attack) | 对敌人造成伤害或施加负面状态 |
| 技能 (Skill) | 提供格挡、抽牌、增益效果等 |
| 能力 (Power) | 被动效果，持续整场战斗 |
| 诅咒 (Curse) | 无法使用，占用牌组空间 |
| 状态 (Status) | 战斗中获得，通常有负面效果 |

### 卡牌稀有度

| 稀有度 | 出现概率 | 特点 |
|--------|----------|------|
| 基础 (Basic/Starter) | - | 初始牌组自带 |
| 普通 (Common) | 约50% | 基础效果 |
| 罕见 (Uncommon) | 约35% | 较强效果 |
| 稀有 (Rare) | 约15% | 强力效果，每层最多见3张 |

### 卡牌费用

- 大多数卡牌费用为0-3点
- X费用：消耗所有剩余能量
- 费用可以被遗物、能力卡等改变

## 遗物系统

### 遗物分类

| 等级 | 说明 |
|------|------|
| 普通 (Common) | 基础加成效果 |
| 罕见 (Uncommon) | 较强效果 |
| 稀有 (Rare) | 强力效果，改变游戏策略 |
| Boss | 击败Boss获得，效果强大 |
| 特殊 | 特定条件获得 |

### 重要遗物

**铁甲战士关键遗物：**
- Burning Blood: 战斗结束时恢复6HP
- Brimstone: 每回合获得2力量，但获得易伤
- Magic Flower: 治疗效果+50%

**沉默猎手关键遗物：**
- Snake Skull: 施加毒素时额外+1毒素
- Wrist Blade: 分身牌伤害+4
- Twisted Funnel: 战斗开始时对敌人施加4毒素

**故障机器人关键遗物：**
- Cracked Core: 战斗开始时充能一个闪电球
- Inserter: 每回合获得一个充能球槽位(最多+2)
- Nuclear Battery: 战斗开始时获得2等离子能量

### 通用构建原则

1. **牌组精简** - 移除基础牌，保留高质量卡牌
2. **核心卡牌** - 确定1-2个核心机制
3. **能量曲线** - 平衡低费和高费卡牌
4. **防御手段** - 确保有稳定的格挡来源

## 游戏机制

### 关键词解释

| 关键词 | 效果 |
|--------|------|
| 力量 (Strength) | 每点增加攻击牌伤害+1 |
| 敏捷 (Dexterity) | 每点增加格挡牌格挡+1 |
| 易伤 (Vulnerable) | 受到伤害+50%，持续X回合 |
| 虚弱 (Weak) | 造成伤害-25%，持续X回合 |
| 毒素 (Poison) | 每回合结束时造成等量伤害，然后-1 |
| 格挡 (Block) | 抵消受到的伤害，下回合移除 |
| 消耗 (Exhaust) | 将牌移出战斗，不再使用 |
| 抽牌 (Draw) | 从抽牌堆将牌加入手牌 |

### 回合流程

1. **回合开始**
   - 触发"回合开始时"效果
   - 获得能量
   - 抽5张牌

2. **玩家行动**
   - 打出卡牌
   - 使用药水
   - 结束回合

3. **回合结束**
   - 触发"回合结束时"效果
   - 弃掉所有手牌
   - 格挡清零

### 敌人行动

- 每个敌人有预设的行动模式
- 可以通过意图图标预测下回合行动
- 攻击、防御、增益、减益等
